Project: ATMSimulator

Previously updated 5/5/22


Application Overview:

My application is a simulation of an ATM machine where the user will be able to withdrawl or deposit money into their "account".


Technology Overview:

I will be using java as my main language for this application.
My file will be saved as a .java file and will run using javac .java command line interface through terminal.


Feature List:

  • A window will first pop up asking for the user to enter in their pin to their bank account.
  • A second window will appear once the user enters in the correct pin. This window will be where
    they will receive further instruction on withdrawing and depositing money into their account.
  • Once the user finishes their transactions, they will select the quit option, where a third
    window will appear with a receipt of their previous transaction history.

Future Feature List (if more time was given):

  • Creating a user sign up list for new clients of the bank.
  • Allowing multiple user accounts to be accessible while in same transaction.
  • Creating a simulation through graphics of the user entering in their "card" into the ATM machine.
  • Potentially adding in voiceover audio within the UI was also something that came to mind.

Basic Technical Features:

  • Implementation of actionListener.
  • Extending JFrame and JPanel for GUI.
  • Utilizing JButton within JPanel.
  • Creating GridLayout for JButtons.

Advanced Technical Features:

  • Using multiple actionListeners for a variation of events.
  • Displaying more than one window in a single use of the program.
